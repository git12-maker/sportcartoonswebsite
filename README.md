# Sportcartoons International Website

A classic static HTML website featuring international sports cartoons.

## Local Development

To run the website locally:

```bash
python3 -m http.server 8000
```

Then open http://localhost:8000 in your browser.

## Deployment

This website can be deployed to:
- GitHub Pages (recommended for static sites)
- Netlify
- Vercel
- Any static hosting service

### GitHub Pages Deployment

1. Push this repository to GitHub
2. Go to Settings > Pages in your GitHub repository
3. Select the main branch as the source
4. Your site will be available at `https://[username].github.io/[repository-name]`

## Structure

- `index.html` - Main homepage
- `prod01.htm` through `prod05.htm` - Language-specific pages (Deutsch, English, Español, Français, Italiano, Nederlands, Português)
- `images/` - Image assets
- `_borders/`, `_derived/`, `_themes/` - FrontPage theme files

## Technologies

- Static HTML
- Microsoft FrontPage (original editor)
